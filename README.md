# Spark
Simple video streaming app built to learn GCP and Full-Stack Development 

# Tech Stack
- TypeScript
- Next.js 13
- Tailwind CSS
- Express.js
- Docker 
- Library: FFmpeg - process videos
- Thunderclient (test endpoints)
- Firebase Auth
- Firebase Functions
- Firebase Firestore
- Google Cloud Storage
- Google Cloud Pub/Sub
- Rest API
- Google Cloud Run
- Artifact registry
- Routing
